\# Omotoso Ebenezer — Personal Website



My live portfolio and public learning logbook, built from scratch with Git and hosted free on GitHub Pages.



\*\*Live site:\*\* https://ebenwebpro.github.io/my-first-website/



\## About



I'm a WordPress and Elementor web designer learning to build world-class

websites using AI-powered workflows. This repo is where I practice the

full professional web workflow — version control, automated checks, and

continuous deployment.



\## Built with



\- Hand-written HTML and CSS

\- Git for version control

\- GitHub Pages for hosting

\- GitHub Actions for automated HTML quality checks



\## How it works



Every change follows the same loop:



1\. Edit `index.html`

2\. Commit the change with Git

3\. Push to GitHub

4\. An Action checks the HTML automatically

5\. GitHub Pages deploys the new version live



\## Status



Actively learning and improving — one commit at a time.
